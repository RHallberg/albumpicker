# MPD Album picker
Simple mpd album picker

**todo:** Add sorting
### Controls:

**HJKL/WASD/Arrow keys:** Navigation

**Q:** Quit

**Space/Enter:** Play album

## Building

Dependencies:
 - mpd
 - libmpdclient
 - odin

```bash
$ odin build . -build-mode:exe
```
